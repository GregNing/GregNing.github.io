將會顯示於X軸上方的數據顯示也就是Y軸 使用`plotOptions => series => dataLabels => enabled`可以將數據直接顯示在折線圖上方 align是位置有點類似padding left概念

```c 將顯示的數據位移
plotOptions:{	
  series: {
  dataLabels: {
  align: 'left',
  enabled: true
  }}}
}
```
